# Phase 6 Exit Criteria

This document acts as the sign-off for Phase 6 (Context & Retrieval).

---

## Completion Checklist

Phase 6 is considered complete when:

1.  **Context is Transparent:** Detection is based on strict, explainable signals (Input/Focus).
    - Verified by `/docs/context_detection.md`

2.  **Filtering is Effective:** High relevance, low noise, strict limits.
    - Verified by `/docs/relevance_filtering.md`

3.  **Assembly is Logical:** Ideas are presented in a coherent structure, not a random bag.
    - Verified by `/docs/contextual_assembly.md`

4.  **Disclosure is Progressive:** Minimal default, expandable details.
    - Verified by `/docs/progressive_disclosure.md`

5.  **Compression is Safe:** Reversible abstraction, no data loss.
    - Verified by `/docs/compression.md`

6.  **Control is Absolute:** User can pin, exclude, and tune.
    - Verified by `/docs/retrieval_controls.md`
